├─ paused_at (TIMESTAMP) - Deployment pause time
├─ completed_at (TIMESTAMP) - Deployment completion time
├─ estimated_duration (INTEGER) - Estimated duration in seconds
├─ rolled_back_at (TIMESTAMP) - Rollback completion time
├─ rolled_back_reason (TEXT) - Rollback reason

└─ updated_at (TIMESTAMP) - Last update time

├─ PRIMARY KEY on id

├─ context (JSONB) - Additional context data
├─ created_at (TIMESTAMP) - Record creation time
└─ updated_at (TIMESTAMP) - Last update time

Indexes: 5
