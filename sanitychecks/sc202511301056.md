- `GET /container-deployments/storage/csi-drivers` - CSI driver status
- `POST /container-deployments/storage/csi-drivers/install` - Install CSI driver
- `GET /container-deployments/storage/classes` - Storage classes
- `POST /container-deployments/storage/classes` - Create storage class

**useDeployment:**
- `GET /container-deployments/k8s/:namespace/deployments` - List deployments
- `POST /container-deployments/k8s/:namespace/deployments` - Create deployment
- `GET /container-deployments/k8s/:namespace/pods` - List pods
- `GET /container-deployments/k8s/:namespace/services` - List services
- `POST /container-deployments/k8s/:namespace/services` - Create service

**useOperations:**

```jsx
// Component usage

  fetchData().catch(() => {
    setNotification({ open: true, message: 'Using simulated data' });
  });

```

### 6.2 Loading Pattern

```jsx
<Button
  disabled={loading}
  startIcon={loading ? <CircularProgress size={16} /> : <ActionIcon />}

</Button>
```

### 6.3 Notification Pattern

```jsx
const [notification, setNotification] = useState({ open: false, message: '' });

// ... in catch block

// ... in render
```
