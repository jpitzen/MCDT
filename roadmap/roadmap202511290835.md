# Cloud Deployment Toolkit Roadmap
## Gap Analysis: Current Tool vs. Complete AWS EKS Deployment

**Date:** November 29, 2025  
**Reference Document:** `issues/AWS Deploy.txt`  
**Current Tool:** Cloud Deployment Toolkit + Container Deployment Service

---

## Executive Summary

The current Cloud Deployment Toolkit provides excellent **Phase 4 (Application Deployment)** capabilities but lacks comprehensive support for **Phase 1 (Prerequisites)**, **Phase 2 (Cluster Configuration)**, and **Phase 3 (Storage Setup)**. This roadmap identifies gaps and prioritizes enhancements to create a complete, full-service EKS deployment tool.

| Risk | Impact | Mitigation |

| IAM permission complexity | High | Provide IAM policy templates, validation |
