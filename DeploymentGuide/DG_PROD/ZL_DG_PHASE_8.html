Application Deployment Overview

Phase 8 deploys the ZL Application components: Document Conversion (Tika), Backend Server, and Web UI.

📌 Version Note: All deployment manifests and container images reference ZL Version 11.1.0 (Build 1140). When deploying different versions, update image tags in all YAML files (zlui-deployment.yaml, zlserver-deployment.yaml, etc.) to match your target build.

🎯 Objectives

  • Create ZLServer ZK EFS Persistent Volumes
  • Deploy Apache Tika service
  • Deploy ZL Server backend application
  • Deploy ZL UI web interface
  • Run ZL ZK Client Init Job (one-time setup)
  • Deploy ZL Search service
  • Verify all application components
  • Complete Phase 8 deployment checklist
