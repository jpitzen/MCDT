EFS CSI Driver Overview

Phase 4 installs the Amazon EFS CSI Driver to provide shared, persistent storage that can be accessed by multiple pods simultaneously.

🎯 Objectives

  • Understand EFS use cases and access points
  • Install and configure EFS CSI Driver with IAM permissions
  • Create EFS file system with security groups and mount targets
  • Mount EFS on EC2/EKS nodes with TLS encryption
  • Create EFS access points for application isolation
  • Deploy StorageClass and persistent volumes
