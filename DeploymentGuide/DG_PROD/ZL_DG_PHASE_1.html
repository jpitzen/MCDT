EKS Cluster Creation Overview

Phase 1 creates the Amazon EKS cluster with managed node groups, configures OIDC provider for IAM roles, and retrieves essential cluster information for subsequent phases.

🎯 Objectives

  • Create EKS cluster with managed node groups
  • Associate IAM OIDC provider for service accounts
  • Retrieve OIDC issuer URL for IAM trust policies
  • Get VPC and subnet information for resource configuration
