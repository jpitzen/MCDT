ZooKeeper Deployment Overview

Phase 7 deploys Apache ZooKeeper to provide distributed coordination services for cluster state management, leader election, and configuration synchronization.

📌 Version Note: ConfigMaps and deployment manifests reference ZL Version 11.1.0 (Build 1140). Verify image tags and configuration values match your target ZL version.

🎯 Objectives

  • Deploy ZooKeeper headless service (zk-hs)
  • Deploy ZooKeeper client service (zk-cs)
  • Create ZooKeeper EFS Persistent Volume and Claim
  • Create zlapp-config ConfigMap (NO ZOO_SERVERS)
  • Create zkclient-config ConfigMap with zkQuorum
  • Deploy ZooKeeper StatefulSet (3 replicas)
  • Run ZooKeeper Config Seed Job (one-time setup)
  • Verify ZooKeeper quorum and pod status
  • Complete Phase 7 deployment checklist
