Appendix D: Security Groups Reference

Reference Updated: Dec 23, 2025

This appendix provides security group configurations for the ZL application deployment on AWS EKS.

📋 Security Group Categories

  • EKS cluster and node security groups
  • Application Load Balancer security groups
  • RDS PostgreSQL database security groups
  • EFS mount target security groups
  • EC2 bastion host security groups
  • Custom security groups for specific services

☸️ EKS Security Groups

EKS Cluster Security Group

Name: zl-eks-cluster-sg
Description: Security group for EKS cluster control plane
VPC: zl-vpc

Inbound Rules:

TypeProtocolPort RangeSourceDescription
All trafficAllAllzl-eks-nodes-sgAllow all traffic from worker nodes
HTTPSTCP44310.0.0.0/16Allow HTTPS from VPC

Outbound Rules:

TypeProtocolPort RangeDestinationDescription
All trafficAllAll0.0.0.0/0Allow all outbound traffic
