Appendix C: IAM Policies Reference

Reference Updated: Dec 23, 2025

This appendix provides complete IAM policy documents used for ZL application deployment on AWS EKS.

📋 Policy Categories

  • EKS cluster and node policies
  • CSI driver policies (EBS, EFS, S3)
  • Load balancer controller policies
  • ECR and container registry policies
  • CloudWatch and monitoring policies
  • Custom policies for specific services

☸️ EKS Cluster Policies

EKS Cluster Administrator Policy

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "eks:*"
            ],
            "Resource": "*"
        },
        {
            "Effect": "Allow",
            "Action": [
                "iam:PassRole"
            ],
            "Resource": "*",
            "Condition": {
                "StringEquals": {
                    "iam:PassedToService": "eks.amazonaws.com"
                }
            }
        }
    ]
}

🔑 Trust Policies for IRSA

OIDC Trust Policy Template

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": {
                "Federated": "arn:aws:iam::ACCOUNT-ID:oidc-provider/oidc.eks.us-east-1.amazonaws.com/id/OIDC-PROVIDER-ID"
            },
            "Action": "sts:AssumeRoleWithWebIdentity",
            "Condition": {
                "StringEquals": {
                    "oidc.eks.us-east-1.amazonaws.com/id/OIDC-PROVIDER-ID:aud": "sts.amazonaws.com",
                    "oidc.eks.us-east-1.amazonaws.com/id/OIDC-PROVIDER-ID:sub": "system:serviceaccount:NAMESPACE:SERVICE-ACCOUNT-NAME"
                }
            }
        }
    ]
}
