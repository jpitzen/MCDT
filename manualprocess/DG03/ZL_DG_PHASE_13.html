Security Scanning Overview

Phase 13 covers container vulnerability scanning using both local (Grype) and cloud (AWS Inspector) tools to ensure container images are secure before production deployment.

🎯 Objectives

  • Understand security scanning tools and methodologies
  • Scan container images for known CVEs using Grype
  • Fetch vulnerability findings from AWS Inspector
  • Use interactive scanner tool to load and analyze results
  • Compare scan results between different tools
  • Generate remediation plans for critical vulnerabilities
