Phase 6: Database Configuration

Step 1 of 9
Database Configuration Overview

Phase 6 creates an Amazon RDS SQL Server Express instance in the same VPC as the EKS cluster for application data storage.

🎯 Objectives

  • Gather VPC and subnet information
  • Create DB subnet group for RDS
  • Configure RDS security group
  • Create RDS SQL Server Express instance
  • Get RDS endpoint for connection
  • Create application database user
  • Configure Kubernetes database ConfigMap and Secret
  • Verify database connectivity via SSM
